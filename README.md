# AG x HN | Freight Dispatch Training

A premium, responsive landing page for remote freight dispatch training. This project features a modern "Collaboration" design with a sleek, gold-accented aesthetic.

## 🚀 Recent Updates
- **Flat Structure:** Moved all assets to the root directory for simpler pathing and hosting.
- **Hero Section:** Refined the "Team" image (gg.png) with a custom bottom-fade mask.
- **Branding:** Integrated the "AG x HN" collaboration logo in the navigation bar.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Custom Variables & Backdrop Filters)
- Responsive Design (Flexbox)

## 📁 Project Structure
- `index.html` - The main landing page.
- `AG.png` - AG Logo (Gold).
- `HN.png` - HN Logo.
- `gg.png` - Main Hero image (Team + Blob).
- `style.css` (or internal CSS) - Custom styling for gradients and layout.

## 🔧 Installation / Local Setup
1. Clone the repository or download the files.
2. Ensure all images (`AG.png`, `HN.png`, `gg.png`) are in the **same folder** as `index.html`.
3. Open `index.html` in any modern web browser.
